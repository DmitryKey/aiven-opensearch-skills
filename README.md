# Aiven for OpenSearch Agent Skill

Agent skill for working with [Aiven for OpenSearch](https://aiven.io/docs/products/opensearch/get-started): create and manage services, connect from code or cURL, manage indices, run search, and configure backups. Covers the Aiven API, `avn` CLI, and Terraform.

## Installation

```bash
npx skills add DmitryKey/aiven-opensearch-skills
```

To install only this skill from a repo that has multiple skills:

```bash
npx skills add DmitryKey/aiven-opensearch-skills --skill aiven-opensearch
```

## When to use

Use this skill when you or your agent:

- Create or connect to an Aiven for OpenSearch service
- Manage indices (create, list, delete) or write OpenSearch queries against Aiven
- Use the Aiven API, `avn` CLI, or Terraform for OpenSearch
